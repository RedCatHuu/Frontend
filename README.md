## 環境
* ruby 3.1.2
* rails 6.1.7
* jQuery 1.12.4
* IED cloud9

## 導入方法
$ git clone git@github.com:RedCatHuu/Frontend.git
$ rails db:migrate
$ rails db:seed
$ yarn install
$ bundle install
$ rails s

## APIは取得できませんでした。
$ yarn install
$ yarn migration:run
$ yarn build
$ yarn start
